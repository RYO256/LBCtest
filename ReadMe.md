# Clean Architecture Demo App
This is a clean architecture app example built with

- Coroutines
- MVVM
- Extension Functions
- Dagger Hilt
- Retrofit
- Room
- Navigation Components
- Cache Strategy (Repository Pattern)
- Coroutines Flow
- ViewBinding
